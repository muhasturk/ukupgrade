Stands for "*Ubuntu Kernel Upgrade*"   

## Why?
It provides us to upgrade linux kernel to latest version for Ubuntu/Mint and derivatives. Based on [this archive](http://kernel.ubuntu.com/~kernel-ppa/mainline/)
